Agar Foto Permanent:
  1. Upload foto dan assign ke paket
  2. Klik "Generate Code"
  3. Copy kode yang muncul
  4. Edit file index.html di repository
  5. Replace bagian bouquetPrices = { dengan kode baru
  6. Commit & push ke GitHub
Cara Upload ke GitHub:
  1. Klik "Export ke GitHub" setelah upload foto
  2. Copy link yang muncul
  3. Buka GitHub repository Anda
  4. Upload foto ke folder /images/
  5. Copy URL foto dan paste ke input URL di bawah
